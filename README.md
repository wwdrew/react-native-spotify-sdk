# @wwdrew/react-native-spotify-sdk

React Native wrapper for iOS and Android Spotify SDKs

## Installation


```sh
npm install @wwdrew/react-native-spotify-sdk
```


## Usage


```js
import { multiply } from '@wwdrew/react-native-spotify-sdk';

// ...

const result = multiply(3, 7);
```
